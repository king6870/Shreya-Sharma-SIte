# Shreya Sharma Achievement Website

One-page portfolio website for Shreya Sharma that highlights academics, leadership, athletic achievements, volunteering, and a featured student-run tennis coaching program.

## Design Direction

- Primary color: Sapphire blue
- Accent color: Light yellow
- Typography: Georgia
- Site type: Single-page with section-based smooth-scroll navigation

## Project Structure

- index.html: Semantic one-page content and section anchors
- styles.css: Theme tokens, layout, responsive design, timeline, and components
- script.js: Active nav state and mobile menu behavior

## Local Preview

Open index.html directly in a browser, or use VS Code Live Server.

## Deploy to Vercel

1. Push this project to a GitHub repository.
2. In Vercel, click New Project and import the repository.
3. Keep Framework Preset as Other.
4. Deploy.
